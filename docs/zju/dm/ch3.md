# Ch3. Algorithms

## Growth of Functions

- Big O - $O$
    - for constants $C$ and $k$, when $x\gt k$, $\lvert f(x) \rvert \le C \cdot \lvert g(x) \rvert$
- Big Omega - $\Omega$
    - for constants $C$ and $k$, when$x\gt k$, $\lvert f(x) \rvert \ge C \cdot \lvert g(x) \rvert$
- Big Theta - $\Theta$
    - $f(x)=O(g(x))$ and $f(x)=\Omega (g(x))$
